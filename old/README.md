# Vincent Picouet - Personal Website

Professional website showcasing research, engineering work, and side projects.

## Structure

- **Research & Engineering**: Instrument science, data analysis, systems engineering
- **Side Projects**: Social science (MSO), astrophotography
- **CV**: Direct link to PDF
- **Contact**: Email contact information

## Local Development

Open `index.html` in a browser to view locally.

## Deployment

This site is automatically deployed via GitHub Pages at `https://vpicouet.github.io`

To update:
```bash
git add .
git commit -m "Update website"
git push origin main
```

Changes will be live within a few minutes.

## Features

- Responsive design
- Image lightbox (click any image to view full size)
- Smooth scrolling navigation
- Fade-in animations on scroll
- Clean, scientific aesthetic

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling
- `script.js` - JavaScript interactions
- `image/` - Image assets
- `csv/` - CV and documents
